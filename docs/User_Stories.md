# User Stories
## VK-Book Android

**Версия:** 1.0  
**Дата:** Февраль 2026

## US-001: Поиск арматуры (Оператор)

**Как** оператор реакторной установки ВК-50  
**Я хочу** быстро найти арматуру по номеру или названию  
**Чтобы** выполнить проверку оборудования без задержек

### Контекст

Операторы ежедневно работают с сотнями единиц арматуры. Быстрый поиск критически важен для эффективной работы.

### Критерии приемки

- [ ] Поиск выполняется менее чем за 5 минут
- [ ] Поиск работает по номеру арматуры
- [ ] Поиск работает по названию арматуры
- [ ] Поиск работает по характеристикам
- [ ] Результаты отображаются с координатами на схеме
- [ ] Поиск работает в офлайн-режиме
- [ ] Результаты можно отсортировать и отфильтровать

### Приоритет: Высокий

### Метрики успеха

- Время поиска: < 5 минут (было: десятки минут)
- Точность результатов: 100%
- Доступность в офлайн-режиме: 100%

## US-002: Просмотр графиков проверок (Инженер)

**Как** инженер по контролю оборудования  
**Я хочу** видеть график проверок с подсветкой текущего часа  
**Чтобы** не пропустить важные замеры

### Контекст

Инженеры должны выполнять проверки оборудования по строгому графику. Пропуск замеров может привести к серьезным последствиям.

### Критерии приемки

- [ ] График отображается с 36-часовой шкалой времени
- [ ] Текущий час подсвечивается визуально
- [ ] Активные задачи выделяются цветом
- [ ] Напоминания срабатывают точно в указанное время
- [ ] Личные и служебные задачи разделены
- [ ] Можно настроить правила повторения напоминаний
- [ ] График загружается из Excel/CSV файла

### Приоритет: Высокий

### Метрики успеха

- Исключение пропусков замеров: > 90%
- Точность напоминаний: 100%
- Визуальная индикация работает в реальном времени

## US-003: Работа со схемами в офлайн-режиме (Обходчик)

**Как** обходчик оборудования  
**Я хочу** просматривать технические схемы без интернета  
**Чтобы** работать в экранированных зонах

### Контекст

Обходчики работают в цехах и подвальных помещениях, где отсутствует доступ к интернету. Доступ к схемам критически важен для выполнения работы.

### Критерии приемки

- [ ] Схемы загружаются в локальный кэш при синхронизации
- [ ] Просмотр схем работает без интернета
- [ ] Поддержка масштабирования и навигации по схеме
- [ ] Быстрый доступ к связанным схемам
- [ ] Отображение маркеров арматуры на схемах
- [ ] Оптимизированный рендеринг больших PDF файлов

### Приоритет: Критический

### Метрики успеха

- Доступность схем в офлайн-режиме: 100%
- Время открытия схемы: < 3 секунды
- Поддержка файлов размером до 50 МБ

## US-004: Управление расписанием смен (Персонал)

**Как** сменный персонал  
**Я хочу** видеть расписание смен на годы вперед  
**Чтобы** планировать личное время

### Контекст

Сменный персонал работает по сложному графику сменности. Необходимо планирование на длительный период (до 2030 года).

### Критерии приемки

- [ ] Календарь отображает график смен на год
- [ ] Поддержка планирования до 2030 года
- [ ] Автоматический расчет графика в обе стороны
- [ ] Визуализация паттернов смен
- [ ] Заметки к дням для дополнительной информации
- [ ] Экспорт/импорт расписания в JSON формате
- [ ] Расчет сдвигов с использованием кэша для оптимизации

### Приоритет: Средний

### Метрики успеха

- Расчет графика: < 1 секунда
- Поддержка планирования: до 2030 года
- Точность расчета: 100%

## US-005: Редактирование данных арматуры (Администратор)

**Как** администратор системы  
**Я хочу** редактировать координаты арматуры и загружать обновления  
**Чтобы** поддерживать актуальность данных

### Контекст

Администраторы должны поддерживать актуальность данных об арматуре. Редактирование координат и загрузка обновлений критически важны для работы системы.

### Критерии приемки

- [ ] Редактирование координат арматуры с валидацией
- [ ] Загрузка обновлений на сервер
- [ ] Проверка целостности файлов перед загрузкой
- [ ] Экспорт данных в Excel с сохранением форматирования
- [ ] Доступ только для администраторов (защита паролем)
- [ ] Индикация статуса загрузки
- [ ] Обработка ошибок при загрузке

### Приоритет: Средний

### Метрики успеха

- Валидация данных: 100%
- Проверка целостности файлов: 100%
- Безопасность доступа: только администраторы

## Резюме

Все пользовательские истории направлены на решение критических проблем операционной деятельности на объекте ВК-50:

1. **Сокращение времени доступа к данным** (US-001, US-003)
2. **Исключение ошибок и пропусков** (US-002)
3. **Планирование и управление** (US-004, US-005)

Каждая история имеет четкие критерии приемки и метрики успеха для измерения эффективности решения.
